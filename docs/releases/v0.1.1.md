# Spheni v0.1.1 Release Notes

## First Public Release

`v0.1.1` is the first public release of Spheni: a tiny CPU-first, in-memory vector search library in C++ with Python bindings.

## Core Features

- Indexes: `Flat` and `IVF`
- Metrics: `L2` and `Cosine`
- Storage: `F32` and `INT8` (scalar quantization)
- Operations: `add`, `search`, `search_batch`, `train`, `save`, `load`

### Key behavior

- Explicit IVF training lifecycle (`train()` before IVF search)
- Query-time IVF control via `nprobe`
- Persistence support for saving/loading engine state
- Python bindings for direct NumPy-based usage

## Compatibility and Packaging

- Python package published as `spheni` on PyPI
- Cross-platform wheel support for:
  - Linux
  - macOS
  - Windows
- Portable wheel defaults (no `-march=native` in package builds)
- OpenMP-enabled package builds for parallel query performance
