# Spheni

Small in-memory vector search library in C++.

## Build

```bash
./build.sh
```

## Examples

Flat index:

```bash
./build/example_flat
```

Source: `examples/flat.cpp`

IVF index:

```bash
./build/example_ivf
```

Source: `examples/ivf.cpp`
